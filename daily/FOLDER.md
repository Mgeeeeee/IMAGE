# daily 目录

一旦我所属的文件夹有所变化，请更新我。

`daily/` 只写日常，不聊工作。  
记录生活、心情与片段记忆。  
保持轻松与真实的语气。  
每次记录需标注日期与时间。  

触发协议：新增/移动/删除/重命名文件 → 更新本文件 → 同步上一级目录索引。

文件清单（名字 / 地位 / 功能）：
- `FREEWRITE.md` — 地位：日常随笔；功能：日常与记忆记录。
- `FOLDER.md` — 地位：目录索引；功能：列出 daily 内容与职责。
