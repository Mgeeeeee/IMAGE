# docs 目录

一旦我所属的文件夹有所变化，请更新我。

`docs/` 只存放文档，不参与运行。  
覆盖使用说明、交接、规范与个人记录。  
文档更新需与代码变更同步。  

触发协议：新增/移动/删除/重命名文件 → 更新本文件 → 同步上一级目录索引。

文件清单（名字 / 地位 / 功能）：
- `README.md` — 地位：使用说明；功能：启动与流程。
- `HANDOVER.md` — 地位：交接文档；功能：架构与维护要点。
- `REQUIREMENTS.md` — 地位：规范基线；功能：协作与技术栈约束。
- `REFLECTIONS.md` — 地位：学习记录；功能：设计心得。
- `FREEWRITE.md` — 地位：自由记录；功能：叨叨念与记忆记录。
- `llms.txt` — 地位：API 索引；功能：外部文档链接集合。
- `FOLDER.md` — 地位：目录索引；功能：列出 docs 内容与职责。

最近更新：2026-01-02，加入文档一致性检查流程。
